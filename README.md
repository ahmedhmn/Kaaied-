اما الآن فأنا وبفضل الزملاء والأصدقاء تعلمت شيئاً فشيا
فتعلمت القليل ولكن بمساعدتهم لي سأعتلي القمه
فدائما وابدا من اراد مساعدتك ينسب له الفضل ولا ينكر فضله
